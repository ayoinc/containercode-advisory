# ContainerCode Advisory - Professional Logo Guidelines

## New Professional Design Philosophy

The ContainerCode Advisory brand has been redesigned to reflect the standards of top-tier management consultancies and technology leaders. Our new visual identity embodies professionalism, expertise, and trustworthiness whilst maintaining technological relevance.

## Logo Variations

### Primary Logo (Horizontal)
- **File**: `containercode-logo-horizontal.svg`
- **Dimensions**: 1200x300px
- **Use**: Website headers, business cards, letterheads, presentations
- **Features**: Professional hexagonal icon with container architecture pattern

### Icon Only
- **File**: `containercode-icon.svg`
- **Dimensions**: 200x200px
- **Use**: Favicons, social media avatars, app icons, mobile applications
- **Features**: Standalone professional icon with background circle

## Design Concept

The new logo represents:
- **Professional Excellence**: Clean, sophisticated design matching consultancy standards
- **Technical Expertise**: Hexagonal pattern symbolising structured, systematic approach
- **Container Architecture**: Internal grid pattern representing containerised solutions
- **Integration**: Connection lines showing system integration and collaboration
- **Trustworthiness**: Professional typography and refined colour palette

- **Usage**: Code snippets, technical references, data displays

## Usage Guidelines

### Professional Standards ✅
- Use on light backgrounds for maximum contrast and readability
- Maintain clear space equal to the height of the icon around the logo
- Scale proportionally to maintain aspect ratio and visual integrity
- Use the horizontal version for headers and wide applications
- Use the icon version for square applications and small sizes

### Brand Consistency ✅
- Always use the approved colour palette
- Maintain consistent typography across all materials
- Ensure sufficient contrast for accessibility compliance
- Use professional, British English in all communications
- Follow the visual hierarchy established in the design system

### Forbidden Practices ❌
- Don't use on dark backgrounds without proper contrast adjustments
- Don't stretch, skew, or distort the logo in any way
- Don't modify the colour scheme or create alternative versions
- Don't separate elements or rearrange the logo components
- Don't use low-resolution versions for high-quality applications
- Don't overlay the logo on busy or distracting backgrounds

## Application Standards

### Digital Applications
- **Minimum Width**: 240px for horizontal logo, 48px for icon
- **File Format**: SVG (preferred), PNG for raster applications
- **Background**: Light colours with sufficient contrast
- **Spacing**: Clear space equivalent to logo height

### Print Applications
- **Minimum Width**: 60mm for horizontal logo, 12mm for icon
- **File Format**: Vector (SVG, EPS) for scalability
- **Print Quality**: 300 DPI minimum for raster versions
- **Colour Mode**: CMYK for print, RGB for digital

### Business Applications
- **Business Cards**: Horizontal logo, minimum 40mm width
- **Letterheads**: Top left positioning, 50-60mm width
- **Presentations**: Consistent placement, readable at distance
- **Signage**: Vector formats for scalability
